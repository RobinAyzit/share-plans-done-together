# Product Overview

DoneTogether is a collaborative task planning and tracking application that enables real-time coordination between friends and teams.

## Core Features

- Real-time collaborative task lists (plans) with progress tracking
- Multi-user support with role-based access (owner, editor, viewer)
- Task completion with photo evidence and reactions
- Social features: friend requests, comments, likes, emoji reactions
- Plan categorization and color coding
- Recurring tasks (daily, weekly, monthly, yearly)
- Task deadlines and reminders
- Multi-language support (English, Swedish, Turkish)
- Push notifications via Firebase Cloud Messaging
- PWA with Android support via Capacitor

## User Experience

- Dark/light theme support
- Confetti celebration on plan completion
- Image compression for task photos
- Invite system with shareable codes
- Auto-cleanup of completed plans after 30 days
- Progress visualization with percentage bars

## Authentication & Data

- Google OAuth authentication via Firebase
- Firestore for real-time data synchronization
- User profiles with friends list and FCM tokens
- Plan ownership with member management
